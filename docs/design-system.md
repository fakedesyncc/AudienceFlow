# AULA · Дизайн-система «Warm Signal»

> Единый источник правды по визуальному языку продукта **AULA** (платформа
> *AudienceFlow*). Этот документ — контракт: и веб-консоль (`services/web`), и
> desktop-клиент (`services/desktop-client`) обязаны использовать одни и те же
> токены. Не хардкодьте цвета мимо этой таблицы.

AULA — система видеоаналитики посещаемости аудиторий. Интерфейс должен читаться
как спокойный, тёплый и премиальный: много воздуха, мягкие тени, скруглённые
карточки, единственный характерный акцент. Тёплый янтарь = присутствие людей и
«под нагрузкой»; глубокий эспрессо = серьёзная инженерная основа; холодный
слейт-тил = данные и измерения (графики, вторичная информация).

## 4. Пространство, радиусы, тени, движение

**Шаг сетки — 4px.** Разрешённые интервалы: `4 8 12 16 20 24 32 40`.

| Токен | Значение |
| --- | --- |
| `--r-sm` | `8px` (поля, мелкие чипы) |
| `--r-md` | `12px` (кнопки, иконки-контейнеры) |
| `--r-lg` | `16px` (карточки, панели) |
| `--r-xl` | `22px` (hero/login-оболочка) |
| `--r-pill` | `999px` (бейджи, прогресс) |

Тени тёплые (тон эспрессо, не чёрные):

| Токен | Значение |
| --- | --- |
| `--shadow-sm` | `0 1px 2px rgba(42,33,28,.06)` |
| `--shadow-md` | `0 1px 2px rgba(42,33,28,.05), 0 10px 28px rgba(42,33,28,.07)` |
| `--shadow-lg` | `0 24px 60px rgba(42,33,28,.16)` |
| `--ring` | `0 0 0 3px rgba(210,105,30,.30)` (фокус) |

Бренд-градиент (login/hero/иконка):
`linear-gradient(135deg, #2A211C 0%, #6E340C 55%, #D2691E 100%)`.

**Движение.** Переходы `140ms` (hover) / `200ms` (появление),
`cubic-bezier(.2,.7,.2,1)`. Обязательна поддержка:

```css
@media (prefers-reduce-motion: reduce), (prefers-reduced-motion: reduce) {
  *, *::before, *::after { animation-duration: .001ms !important; transition: none !important; }
}
```

## 5. Компоненты (правила)

- **Карта / панель:** `--surface`, `--r-lg`, `1px solid --line`, `--shadow-md`,
  внутренний отступ `16–20px`. Без жёстких чёрных рамок.
- **Метрика:** иконка в мягком тинте акцента (`--amber-100` фон, `--amber-700`
  символ), подпись `--ink-600` (overline), значение 30/800 `--ink-900`.
  Левый цветной кант заменяем на мягкий тинт — без «полосок-светофора».
- **Primary-кнопка:** фон `--amber-600`, текст белый, `--r-md`, вес 700, hover
  `--amber-700`, focus `--ring`. Иконка слева, gap `8px`.
- **Secondary-кнопка:** белый фон, `1px solid --line`, текст `--amber-700`, hover
  фон `--sand`.
- **Поле ввода:** белый фон, `1px solid --line`, `--r-sm`, min-height `40px`,
  focus — граница `--amber-500` + `--ring`.
- **Бейдж состояния:** pill, семантические токены из §2. Всегда иконка + текст.
- **Прогресс наполняемости:** трек `--sand`, заливка по статусу, высота `8–10px`,
  `--r-pill`.
- **Таблица:** шапка `--sand`, `--ink-600` 12/700 uppercase; строки на `--surface`,
  hover — тинт `--amber-50`, выбранная строка — `--amber-100` + левый кант
  `--amber-500`.
- **Навигация (sidebar):** белая поверхность на `--canvas`, активный пункт —
  `--amber-100` фон + `--amber-700` текст/иконка; левый кант `--amber-500`.
- **Пустые / ошибочные состояния:** иконка + короткий текст + **подсказка к
  действию** (кнопка или ссылка), не только сообщение.

## 6. Доступность (обязательно)

- Контраст текста ≥ WCAG **AA** (см. заметки о контрасте в §2).
- Видимый фокус клавиатуры на всех интерактивных элементах (`--ring`), не
  убирать `outline` без замены.
- Вся навигация доступна с клавиатуры; логичный порядок табов; `aria-label` на
  иконочных кнопках; таблицы с семантическими `th`.
- Уважать `prefers-reduced-motion` (см. §4).
- Не передавать смысл только цветом — дублировать текстом/иконкой (бейджи).

## 7. Иконки и логотип

- **Знак AULA** — монограмма «A»: две ножки-арки (аудитория) + перекладина как
  «линия учёта» с узлами-датчиками на концах (вход/выход). Файлы:
  - `services/web/public/favicon.svg` — компактный знак;
  - `services/web/public/brand/aula-icon.svg` — плитка приложения (градиент);
  - `services/web/public/brand/aula-icon-maskable.svg` — maskable (safe-zone);
  - `services/web/public/brand/aula-mark.svg` — моно-знак (янтарь, прозрачный фон);
  - PNG-растры `aula-icon-192/512`, `aula-maskable-512`, иконка окна desktop.
- **Логотип в интерфейсе** = знак `aula-mark.svg` + текст `AULA` + подпись
  `by AudienceFlow`. Не растягивать, не менять цвет знака вне палитры.
- **Продуктовые иконки** — единый штриховой/сплошной набор одной метрики
  (web: lucide; desktop: `IconGlyph`). Символ иконки на акценте — `--amber-700`.
- **theme-color / манифест:** `#D2691E`; `background_color` браузерного шелла
  `#FAF7F2`.
